# Hello World

A simple quickstart script demonstrating how AutomataHub discovers and executes scripts.

## What It Does

1. Prints a greeting and the current date
2. Runs a 3-step loop with 1-second delays
3. Exits successfully

## Structure

```
hello-world/
├── config.json   # Script metadata (name, description, entry point)
├── main.sh       # The executable script
└── README.md     # This file
```

Use this folder as a template when creating your own scripts.
